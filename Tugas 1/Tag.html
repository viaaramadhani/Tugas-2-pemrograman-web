Penulisan TAG

  1. Tag Pembuka
  2. Tag Penutup
  3. Single Tag

    Contoh: <br />, <hr />, <input />, <img />

Tag Heading

Tag heading digunakan untuk menuliskan judul & sub-judul. Tag heading terbagi menjadi 6, yaitu:

  1. Heading 1 (H1)
  2. Heading 2 (H2)
  3. Heading 3 (H3)
  4. Heading 4 (H4)
  5. Heading 5 (H5)
  6. Heading 6 (H6)

Tag Paragraf, HR, dan BR

  • <p>isi paragraf</p>
  • <br /> : breakline
  • <hr /> : horizontal line

Font Tag

Contoh:

<font color="blue" size="7" face="arial">Test</font>

Tag lain untuk manipulasi font:

  • <b>tulisan tebal</b>
  • <i>tulisan miring</i>
  • <u>tulisan bergaris bawah</u>
  • <sub>subscript</sub>, contoh: ini contoh tulisan subscript
  • <sup>superscript</sup>, contoh: ini contoh tulisan superscript

Image Tag

Contoh:

<img src="logo-untad.jpg" />

Tag image memiliki atribut:

  • src: lokasi gambar yang akan ditampilkan
  • width: ukuran lebar gambar
  • height: ukuran tinggi gambar
  • alt: deskripsi tentang gambar
  • title: judul gambar

Anchor/Hyperlink Tag

Digunakan untuk menghubungkan (linking) teks dan gambar ke halaman lain atau bagian tertentu dari halaman yang sama.

Contoh:

  1. Link ke halaman website lain
    • Silahkan tekan "Link menuju ke website Tailwind CSS"
  2. Link ke file lain dalam satu website
    • Silahkan tekan "Ini akan menuju ke landing page"
  3. Penanda bagian halaman
    • Ini adalah alinea pertama
  4. Link ke file yang sama
    • Ini akan kembali ke bagian awal
